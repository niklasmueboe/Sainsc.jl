## Installation

The package can be installed with the Julia package manager.
From the Julia REPL, type `]` to enter the `Pkg` REPL mode and run:

```
pkg> add StereoSSAM
```

Or, alternatively, via the `Pkg` API:

```julia
using Pkg
Pkg.add("StereoSSAM")
```
