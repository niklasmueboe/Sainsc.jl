Sainsc.jl

Sainsc.jl - Segmentation-free analysis of in situ capture data[1] - is, as the name states, a tool to analyse in situ capture-based spatial transcriptomics data. Actually, it can also be used to analyse imaging-based datasets. For a more thorough background please refer to the original publication or just follow along the provided examplary analysis.

A more batteries included[2] version of this package is available in Python/Rust. If you are interested have a look at https://github.com/HiDiHlabs/sainsc.

Installation

The package can be installed with the Julia package manager. From the Julia REPL, type ] to enter the Pkg REPL mode and run:

pkg> add Sainsc

Or, alternatively, via the Pkg API:

using Pkg
Pkg.add("Sainsc")
